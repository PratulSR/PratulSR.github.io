# React Code 

## required libs (install with npm)

- typescript
- react
- axios
- swiper
- react-dom
- react-router-dom
- frappe-gantt-react
- react-material-ui-carousel
- @mui/material 
- @emotion/react
- @emotion/styled
- @fontsource/roboto
- @mui/icons-material

  ## HOW TO OPEN THE ENVIRONEMNT
  Use the terminal enter into the project file.
  ````
  cd gantt-vite-react
  ````
  Help you install all dependencies
  ````
  npm i
   ````
  The swiper package sometimes can not be add using the npm installtion so it is important to add the swiper.
   ````
  npm install swiper
    ````
   Make the project working
   ````
  npm run dev
    ````
